What is Electroencephalography (EEG)?#

Electroencephalography (EEG) [1] is a method to record an electrogram of the spontaneous electrical activity of the brain. The bio potential signals detected by EEG have been shown to represent the postsynaptic potentials of pyramidal neurons in the neocortex and allocortex. It is typically non-invasive, with the EEG electrodes placed along the scalp (commonly called ‚Äúscalp EEG‚Äù) using the International 10‚Äì20 system, or variations of it. Clinical interpretation of EEG recordings is most often performed by visual inspection of the tracing or quantitative EEG analysis

To know more about EEG visit here.

The International 10‚Äì20 system is a standard method for placing EEG electrodes on the scalp in relation to the brain‚Äôs cerebral cortex. Electrode positions are labeled with letters (like Fp, F, T, C, P, O, and Z for midline) and numbers (odd for left, even for right hemisphere). The ‚Äú10‚Äù and ‚Äú20‚Äù refer to the percentage distances between electrodes based on the size of the head. Key reference points like the nasion, inion, and preauricular points are used for accurate placement. The system helps ensure consistent, repeatable, and comparable EEG recordings. More detailed versions like the 10‚Äì10 or 10‚Äì5 systems provide higher resolution by adding extra electrodes.

To know more about International 10‚Äì20 system visit here.

1. Fixed Sampling

1. Program Purpose & Overview

The Fixed Sampling sketch acquires raw EEG/biopotential data from the Brain‚ÄëBioAmp‚Äôs ADC at a user‚Äëdefined, constant rate. By using a hardware timer interrupt rather than a delay loop, it guarantees uniformly‚Äëspaced samples‚Äîcritical for accurate digital filtering, spectral analysis, or machine‚Äëlearning pipelines downstream.

2. How It Works

  1. Initialize the Sensor Pin

    • The sketch sets an Arduino analog input pin (e.g., A0) to read voltage values from the BioAmp sensor.

  2. Set Sampling Rate

    • A timer (using micros() or delayMicroseconds()) ensures that we call analogRead(A0) at a precise interval.

  3. Print Raw Values

    • The user sees raw voltage fluctuations corresponding to brain waves.

  4. Loop Forever

    • The loop() continues indefinitely, constantly reading and printing.

2. EEG Filter

1. Program Purpose & Overview

The EEG Filter sketch acquires raw EEG data from the BioAmp EXG Pill sensor at 256‚ÄØHz and applies a 0.5‚ÄØ‚Äì‚ÄØ29.5‚ÄØHz 4th‚Äëorder Butterworth band‚Äëpass filter (implemented as four biquad sections) to isolate true EEG rhythms. By stripping out slow DC drifts and high‚Äëfrequency noise, you obtain a clean EEG stream ideal for real‚Äëtime visualization, event detection, or further spectral analysis.

2. How It Works

  1. Initialize the Sensor Pin

    • The sketch configures an Arduino analog input pin (e.g., A0) to read voltage values from the BioAmp sensor.

  2. Compute Elapsed time

    • A static past timestamp holds the previous sample‚Äôs microsecond count.

    • present = micros() and interval = present ‚Äì past give the time since the last loop.

    • past is updated to present for the next iteration.

  3. Run the Sample Timer

    • A static timer variable counts down by interval each loop.

    • When timer < 0, it‚Äôs time to take the next sample:

timer += 1000000 / SAMPLE_RATE;  // ‚âà3906‚ÄØ¬µs for 256‚ÄØHz

  1. Acquire Raw Sample

    • Calls analogRead(INPUT_PIN) (e.g. A0) to get the latest ADC count from the BioAmp output.

    • Converts the integer reading to a float sensor_value.

  2. Apply 4th‚ÄëOrder Butterworth Band‚ÄëPass.

    • Passes sensor_value into EEGFilter(input), which implements four cascaded biquad sections.

    • Coefficients (a1, a2, b0, b1, b2) were generated via SciPy‚Äôs butter() and exported by filter_gen.py.

    • Each section maintains two static states (z1, z2), computes the difference‚Äëequation:

x = output ‚Äì a1*z1 ‚Äì a2*z2;
output = b0*x + b1*z1 + b2*z2;
z2 = z1;
z1 = x;

  1. Stream Filtered Output

    • After filtering, Serial.println(signal); sends the clean EEG value to your PC or host.

  2. Loop Forever

    • The sketch never blocks: timing logic and filtering run every ‚âà3.9‚ÄØms (256‚ÄØHz), then immediately repeat.

  • To learn more about filters and how to generate new filters, visit: https://docs.scipy.org/doc/scipy/reference/generated/scipy.signal.butter.html
